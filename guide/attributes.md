# 页面属性方法
包含页面的方法事件和属性

## 通用
| 类型               | 说明                 | 默认值 |
| :----------------- | :------------------- | :----- |
| reset-row-dblclick | 是否禁用页面双击事件 | false  |
| use-list-show-back | 是否添加返回按钮     | false  |
