# 日期工具

```js
this.$tu.utils.tools.formatDate(日期对象缺省为当前日期, 格式化字符串缺省值为yyyy-MM-dd)
this.$tu.utils.tools.string2Date(日期字符串, 分割值缺省值为-)
this.$tu.utils.tools.dateAdd(模式, 数值（1/-1）, 基础日期缺省值为当前日期)
// 模式
this.$tu.constant.DATE_FORMAT.Y
Y: 'y', // 年
Q: 'q', // 季度
M: 'm', // 月
W: 'w', // 周
D: 'd', // 天
H: 'h', // 小时
MI: 'mi', // 分钟
S: 's', // 秒
```
